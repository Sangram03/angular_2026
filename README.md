Download and install from 👉 Node.js official website.

Check installation:

```bash
node -v
npm -v
```

---

### ✅ 2. Install Angular CLI

The Angular CLI helps you create and manage Angular projects.

```bash
npm install -g @angular/cli
```

Verify installation:

```bash
ng version
```

---

### ✅ 3. Create a New Angular Project

```bash
ng new my-app
```

---

### ✅ 4. Navigate into Project Folder

```bash
cd my-app
```

---

### ✅ 5. Run the Angular App

```bash
ng serve
```

Open your browser and go to:

```
http://localhost:4200/
```
